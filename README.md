# Accessible Hamburger Menu 🍔

Este repositorio contiene el código fuente del tutorial paso a paso para construir un menú de navegación tipo "hamburguesa" completamente accesible, estético y funcional. 

El objetivo de este proyecto es demostrar cómo elevar los estándares de desarrollo Frontend utilizando prácticas de accesibilidad (**A11y**) de clase mundial.

## 🚀 Características del Proyecto
*   **Semántica HTML5:** Uso de etiquetas correctas para tecnologías asistivas.
*   **Atributos ARIA:** Implementación de `aria-expanded`, `aria-label` y `aria-controls`.
*   **Diseño Moderno:** Maquetación con Flexbox y CSS Variables.
*   **WCAG Compliant:** Contraste de color optimizado (Nivel AA) y estados de foco visibles.
*   **JS Vanilla:** Lógica ligera para la gestión de estados de accesibilidad.

## 🛠️ Validación Profesional
Este componente ha sido testeado y validado superando con éxito las siguientes pruebas:
1.  **Axe DevTools:** 0 errores detectados en el análisis automático de accesibilidad.
2.  **Navegación por Teclado:** Totalmente operable mediante la tecla `Tab` y `Enter`.
3.  **Lectores de Pantalla:** Verificado para una lectura correcta de estados (abierto/cerrado).

## 💻 Herramientas Utilizadas

*   Chrome DevTools + Axe DevTools
